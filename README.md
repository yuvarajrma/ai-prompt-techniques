# AI Prompt Library — Frontend & Backend

A curated collection of copy-paste-ready, two-liner prompts for building frontend (React) and backend (Node.js) features with AI assistants.

Each prompt follows a simple format: **what to build** + **which design pattern to follow** — giving the AI both the goal and the approach in two concise lines.

## 🚀 How to Use

1. Open the relevant prompt file (frontend or backend).
2. Find the feature or module you need.
3. Copy the two-liner prompt into your AI assistant's chat.
4. Let the AI generate the implementation following the named pattern.

You can use these prompts as-is or tweak them to fit your project's specific stack, naming conventions, or requirements.

---

## 🧩 Prompt Format

Every prompt follows this structure:

> **First line** — Describes *what* to build and the key requirements.
> **Second line** — Names the *design pattern or stereotype* to follow and explains why.

This keeps prompts short enough to drop into any AI chat while giving enough direction to get clean, pattern-driven code back.
